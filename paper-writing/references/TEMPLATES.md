# 项目文件模板

## 1. PROJECT_GUIDE.md

```markdown
# [项目名称] 项目指南

## 项目概述

**研究主题**: [一句话描述研究内容]

**研究问题**: [具体要解决的问题]

**核心贡献**:
1. [贡献1]
2. [贡献2]
3. [贡献3]

## 写作规范

### 术语表

| 术语 | 定义 |
|------|------|
| Term 1 | Definition |
| Term 2 | Definition |

### 缩写表

| 缩写 | 全称 |
|------|------|
| CNN | Convolutional Neural Network |
| LSTM | Long Short-Term Memory |

### 引用风格

- 使用数字引用 [1], [2-5]
- 参考文献按出现顺序排序
- 作者姓名统一格式

### 数据格式

- 精度: 保留2位小数 (0.89)
- 百分比: 95.2%
- 统计显著性: p < 0.05

## 目标期刊/会议

- 名称: [如 IEEE CVPR]
- 模板: [链接]
- 截稿日期: [日期]
- 页数限制: [如 8页]

## 进度追踪

- [ ] 阶段1: 文献调研
- [ ] 阶段2: 结构规划
- [ ] 阶段3: 文献综述
- [ ] 阶段4: 方法实验
- [ ] 阶段5: 正文撰写
- [ ] 阶段6: 修改投稿
```

- [ ] 局限性
- [ ] 未来工作

### 6. Conclusion (0.5页)

**要点**:
- [ ] 贡献总结

**预计完成**: _____

## 里程碑

| 里程碑 | 目标日期 | 状态 |
|--------|----------|------|
| 文献调研完成 | YYYY-MM-DD | ⬜ |

| 投稿 | YYYY-MM-DD | ⬜ |

## 图表清单

| 编号 | 类型 | 标题 | 状态 |

| Fig. 2 | 结果图 | Performance Comparison | ⬜ |
| Table 1 | 数据集 | Dataset Statistics | ⬜ |
