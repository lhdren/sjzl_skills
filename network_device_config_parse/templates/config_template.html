# 网络设备配置文件模板 ## Cisco 路由器配置模板 ```cisco hostname RT-CORE-01 ! no ip domain-lookup ! interface GigabitEthernet0/0 description WAN Interface ip address 202.96.128.1 255.255.255.252 no shutdown ! interface GigabitEthernet0/1 description LAN Interface ip address 192.168.1.1 255.255.255.0 no shutdown ! router ospf 1 network 192.168.1.0 0.0.0.255 area 0 ! line vty 0 4 password cisco login ! end ``` ## Cisco 交换机配置模板 ```cisco hostname SW-ACCESS-01 ! vlan 10 name SALES ! vlan 20 name ENG ! interface GigabitEthernet0/1 switchport access vlan 10 switchport mode access spanning-tree portfast ! interface GigabitEthernet0/2 switchport trunk encapsulation dot1q switchport mode trunk switchport trunk allowed vlan 10,20 ! interface Vlan10 ip address 192.168.10.1 255.255.255.0 no shutdown ! spanning-tree mode rapid-pvst ! end ``` ## Huawei 路由器配置模板 ```huawei sysname RT-CORE-01 # acl number 2000 rule 5 permit source 192.168.1.0 0.0.0.255 # interface GigabitEthernet0/0/0 description WAN Interface ip address 202.96.128.1 255.255.255.252 undo shutdown # interface GigabitEthernet0/0/1 description LAN Interface ip address 192.168.1.1 255.255.255.0 undo shutdown # ospf 1 area 0.0.0.0 network 192.168.1.0 0.0.0.255 # return ``` ## Huawei 交换机配置模板 ```huawei sysname SW-ACCESS-01 # vlan batch 10 20 # interface GigabitEthernet0/0/1 port link-type access port default vlan 10 # interface GigabitEthernet0/0/2 port link-type trunk port trunk allow-pass vlan 10 20 # interface Vlanif10 ip address 192.168.10.1 255.255.255.0 # stp mode rstp # return ``` ## H3C 交换机配置模板 ```h3c sysname SW-ACCESS-01 # vlan 10 name SALES # vlan 20 name ENG # interface GigabitEthernet1/0/1 port access vlan 10 # interface GigabitEthernet1/0/2 port link-type trunk port trunk permit vlan 10 20 # interface Vlan-interface10 ip address 192.168.10.1 255.255.255.0 # stp mode rstp # return ``` ## Juniper 路由器配置模板 ```juniper set system hostname RT-CORE-01 set interfaces ge-0/0/0 description WAN set interfaces ge-0/0/0 unit 0 family inet address 202.96.128.1/30 set interfaces ge-0/0/1 description LAN set interfaces ge-0/0/1 unit 0 family inet address 192.168.1.1/24 set protocols ospf area 0.0.0.0 interface ge-0/0/1 set system login user admin authentication plain-text-password ``` ##
